# azureterraformresources

Project is a simple IaC-Setup using Terraform on Microsoft Azure.

### Topic

The goal is to provision basic cloud infrastructure in Azure using Terraform and understand how Infrastructure as Code works in practice.

### Resources

- Azure Resource Group
- Azure Storage Account
- Blob Storage Container

### Technical Setup 
- Terraform
- Microsoft Azure

------

- Infrastructure as Code (IaC) principles
- Basic Azure cloud resource provisioning
- Terraform workflow (init, plan, apply)
